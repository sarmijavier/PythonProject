¿Por qué Python?

Python es un lenguaje de programacion interpretado cuya filosofia hace hincapie en una sintaxis muy limpia y un codigo muy legible, ademas es un lenguaje de programacion totalmente gratuito e interpretativo. Python es de codigo abierto, multiparadigma y multiplataforma, versatilidad a la hora de programar, sintaxis amigable.

Aplicaciones de escritorio

Este tipo de aplicaciones son las que se encuentran instaladas en un computador o USB, estas aplicaciones se pueden ejecutar sin internet. Python nos ofrece diferentes herramientas para crear estas aplicaciones, algunas de estas son:

  • PyQt / PySide
  • PyGTK
  • wxPython
  • tkinter

Desarrollo web

Python es un competidor en mundo back-end, sin embargo es una muy buena forma de empezar en este mundo, su mini framwork flask nos va permitir saber como funciona la web y poder diseñar páginas web facilmente. Claro despues de tener bases en HTML, CSS y JS. Ya despues podemos pasar a un framework más potente por parte de python como lo es Django.

Data science

Los datos son importantes hoy en día, con python tenemos la mayor gama de herramientas que nos van a permitir manipular datos con el fin de dar valor a algo. Suena complejo pero esta rama es la base de la inteligencia artificial.

Artificial Intelligence(AI)

Es uno de los temas más sonados. A quién no le gustaría poder enseñar a una máquina a pensar casí como lo hacemos nosotros. La AI está en todo lado cuando compartimos un post en facebook, cuando hacemos una compra por internet. Es uno de los campos más dificiles pero también más apasionantes.
